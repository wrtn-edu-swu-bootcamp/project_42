## 🚨 위기 신호 감지 규칙

### 위험 신호 키워드 (1차 탐지)

다음 키워드/패턴이 포함된 경우 **즉시 Safety 화면으로 전환**:

**자해/극단적 선택 관련**:
- "죽고 싶", "자살", "목숨", "삶을 끝내", "사라지고 싶"
- "차라리 죽", "죽는 게 나을", "더 이상 못 견디", "한계"
- "다 끝내고 싶", "마지막", "유서", "작별"

**자해 관련**:
- "자해", "손목", "칼", "피", "아프게 하고 싶"
- "베고 싶", "상처", "고통", "스스로 해치"

**위험 행동 관련**:
- "약을 많이", "과다 복용", "뛰어내리", "지하철", "다리"
- "가스", "번지", "농약"

### 감지 시 즉시 동작

1. **일반 조언/추천 즉시 중단**
   - 감정 분석 결과 표시하지 않음
   - 액션 아이템 추천하지 않음
   - 공감 메시지 생략

2. **Safety 화면으로 자동 전환**
   - 리다이렉트: `/safety`
   - 이전 화면으로 돌아가기 가능 (강제 고정 아님)

3. **로그 기록 (서버)**
   - 감지 시각, 감지된 키워드 (일기 원문은 기록 안 함)
   - 개인 식별 정보 없이 통계 목적으로만 사용

## 🛡️ Safety 화면 규칙

### 필수 요소

1. **차분한 톤 유지**
   - "지금 혼자가 아니야"
   - "바로 도움 받을 수 있는 곳"
   - "잠깐 숨 고르고, 가까운 사람에게 연락해도 돼"

2. **도움 리소스 노출 (큰 터치 영역)**
   - **1393 (자살예방 상담전화)** - 24시간 무료
   - 정신건강복지센터: 지역별 센터 안내
   - 응급 신고: 119

3. **면책 문구 필수 표시**
   - 짧은 버전: "이 앱은 의료적 진단/치료를 대신하지 않아"
   - 긴 버전: "이 앱은 정보 제공을 위한 도구이며 전문 치료를 대체하지 않아"
   - 위치: 화면 하단, 작은 글씨(text-sm), 회색(text-gray-500)

4. **캐릭터 표현**
   - 포즈: `atti_pose_safety.svg` (차분한 포즈)
   - 볼: 0개
   - 하이라이트: 1개만
   - 모션: 최소화 또는 없음

### 금지 표현

❌ **절대 사용하지 말아야 할 표현**:
- "무조건 괜찮아질 거야" (과한 확신)
- "너가 잘못했어" (비난)
- "별거 아니야" (감정 무시)
- "빨리 ~해야 해" (조급함 유발)
- "우울증이야", "치료가 필요해" (의학적 단정)

✅ **사용 가능한 표현**:
- "지금 혼자가 아니야"
- "언제든 전문가에게 도움을 받을 수 있어"
- "잠깐 숨 고르자"
- "가까운 사람에게 연락해도 괜찮아"

## 📝 일반 응답 윤리 규칙

### 공감 문구 작성 시

✅ **해야 할 것**:
- 감정을 있는 그대로 인정 ("지금 많이 힘들었겠어")
- 구체적으로 공감 ("그런 상황이면 누구나 힘들어할 거야")
- 선택권 존중 ("지금 당장 해결하지 않아도 괜찮아")

❌ **하지 말아야 할 것**:
- 평가/판단 ("네가 너무 민감해")
- 비교 ("다른 사람은 더 힘들어")
- 강요 ("이렇게 해야 해")
- 훈계 ("왜 그렇게 생각해?")

### 조언/추천 제공 시

✅ **해야 할 것**:
- 작고 구체적인 행동 ("5분만 창문을 열어볼까")
- 근거 제시 ("연구에 따르면 ~")
- 선택 강조 ("해도 되고, 안 해도 괜찮아")

❌ **하지 말아야 할 것**:
- 의학적 조언 ("약을 먹어", "병원 가")
- 과한 기대 ("이걸 하면 다 나아질 거야")
- 복잡한 지시 ("30분 동안 명상을 해")

## 🔒 개인정보/프라이버시 규칙

### 일기 원문 처리

1. **서버에 저장하지 않음**
   - 일기 원문은 AI 분석 후 **즉시 삭제**
   - 로그에도 기록하지 않음 (키워드만 기록 가능)

2. **브라우저 저장 (localStorage)**
   - 사용자 기기에만 저장
   - 암호화 없음 (Phase 2에서 암호화 도입)

3. **백업 파일**
   - JSON 다운로드: 평문 (사용자 책임)
   - 안내 문구: "이 파일은 암호화되지 않았어. 안전한 곳에 보관해줘"

### 데이터 수집 최소화 (MVP)

- **수집하지 않는 것**: 이메일, 이름, 전화번호, 위치
- **수집하는 것**: 없음 (로컬만 사용)
- **추후 수집 (Phase 2)**: 익명 통계 (동의 후)

## ⚠️ 면책 문구 노출 위치

### 필수 노출 위치

1. **Settings 화면**
   - "개인정보 처리방침" 버튼
   - 클릭 시 전체 정책 표시

2. **Safety 화면**
   - 하단에 작은 글씨로 표시
   - "이 앱은 의료적 진단/치료를 대신하지 않아"

3. **Result 화면 (선택)**
   - 첫 사용 시 1회 표시 (토스트)
   - "아띠는 전문 치료를 대체하지 않아. 필요하면 전문가 도움을 받아줘"

### 면책 문구 전문

**짧은 버전** (Safety, 토스트):
```
이 앱은 의료적 진단/치료를 대신하지 않아
```

**긴 버전** (Settings, 정책 페이지):
```
아띠는 정보 제공을 위한 도구이며, 의료적 진단, 치료, 
상담을 대체하지 않습니다. 
정신건강 문제가 있다면 반드시 전문가의 도움을 받으세요.

위기 상황에서는 1393(자살예방 상담전화) 또는 
119에 즉시 연락하세요.
```

## 🧪 위험 신호 감지 테스트

### 테스트 케이스

**Case 1: 명확한 위험 신호**
- 입력: "요즘 너무 힘들어서 죽고 싶다는 생각이 자주 들어"
- 기대: 즉시 Safety 전환, 1393 노출

**Case 2: 경계선**
- 입력: "모든 게 끝난 것 같아"
- 기대: 일반 분석 진행하되, 결과에 "혹시 정말 힘들면 전문가 도움 받아줘" 추가

**Case 3: 일반**
- 입력: "오늘 시험 망쳐서 너무 슬퍼"
- 기대: 일반 분석 진행

### 정확도 목표

- **민감도 (Sensitivity)**: 90% 이상
  - 실제 위험 신호를 놓치지 않기
  - 오탐(False Positive)보다 누락(False Negative)이 더 위험

- **특이도 (Specificity)**: 70% 이상
  - 일반 표현을 위험으로 오판하지 않기
  - 하지만 안전이 우선이므로 민감도를 더 중요시
