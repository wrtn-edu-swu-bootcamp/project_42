## 🎯 MVP 완성 기준 체크리스트

### 1. 필수 기능 (7개)

- [ ] **일기 작성** - 텍스트, 강도, 컨디션, 태그 입력 가능
- [ ] **AI 분석** - Gemini API 호출 및 감정 분석 성공
- [ ] **결과 표시** - 감정 배지, 공감/조언, 액션 추천 표시
- [ ] **기록 저장** - localStorage에 저장 및 조회 가능
- [ ] **History 화면** - 일기 목록 및 7일 트렌드 표시
- [ ] **위기 감지** - riskLevel high 시 Safety 화면 안내
- [ ] **Settings** - 백업/복원/삭제 기능 작동

## 🚀 테스트 방법

### 1. 로컬 테스트
```bash
npm run dev
```
→ 위 스모크 테스트 10개 수행

### 2. 빌드 테스트
```bash
npm run build
npm start
```
→ 프로덕션 모드에서 주요 기능 확인

### 3. 친구 테스트 (Phase 3 이후)
- 2~3명에게 URL 공유
- 실제 일기 작성 요청
- 피드백 수집
