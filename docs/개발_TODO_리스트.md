# Roadmap Summary (한 눈에 보기)
- Phase 0: 개발 시작을 위한 환경/규칙 확정 + 기본 세팅(환경 준비, 폴더 구조, 안전 규칙)
- Phase 1: MVP 핵심 기능 완성(일기 입력, AI 분석, 결과/저장/History, Safety, 상태 UI)
- Phase 2: Beta 안정화/품질 강화(검증, 카탈로그 정비, 백업/복원, 테스트 운영)
- Phase 3: Launch 배포/검증/피드백(배포, 스모크 테스트, KPI 운영)
- Phase 4: Hardening/Scale 고도화(클라우드 DB, 로그인, 암호화, 성능/근거/접근성 강화)
