## 💡 핵심 컨셉 요약

### "아띠"란?

**이름 의미**
- 순우리말 "아띠" (친한 친구) + AI (인공지능)
- 친근함 + 똑똑함의 이중 의미

**한 줄 설명**
> 오늘의 감정을 말로 정리하고, 근거 있는 작은 행동을 제안받는 감정 일기 웹앱

**타겟**
- 10대~30대 (혼자 감정을 처리해야 하는 시간이 많은 세대)

**차별점**
- 130+ 세밀한 감정 라벨
- 심리상담학 이론 기반 분석 (CBT, Rogers, Maslow 등)
- 근거 있는 액션 추천 (음악, 꽃, 활동 등)
- 저작권 안전한 음악 추천 (가수+곡명+가사+이유)

## 🛠️ 기술 스택 (한눈에)

```
프론트엔드:  Next.js 14 + TypeScript + Tailwind CSS + shadcn/ui
백엔드:      Next.js API Routes
AI:          OpenAI API (GPT-3.5-turbo)
저장소:      localStorage (MVP) → Supabase (Phase 2)
배포:        Vercel (무료)
```
